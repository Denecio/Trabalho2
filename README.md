This website shows the information about ENED, the National Meeting of Design Students.

The website is divided into 4 sections: Home, Packages, Program and Team.

The Home section is the main page of the website, where you can find a brief description of the event and the date it will take place.

The Packages section shows the different types of tickets that can be purchased for the event, as well as the benefits of each one.

The Program section shows the schedule of the event, with the date and time of each activity.

The Team section shows the members of the organizing team, with a brief description of each one.
